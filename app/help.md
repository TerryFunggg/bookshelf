# Command
## add book   
command: `add "{book name}"`
alias:   `-a`
Example: 
- `books add "Data Structure in JAVA"`

## Fuzzy List/Search book
command: `list {keywords}`
alias: `-l`
Example:
- `books list`  // given 10 results group by status and date.
- `books list java`

## Update book
`update`  `-u`    : update books

## Remove book
`delete`  `-d`    : remove books
